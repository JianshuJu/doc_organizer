# <p>

## 概述

`<p>` 元素用于定义段落，是 HTML 中最基本的文本块级元素之一。它表示一段文本内容的语义分组，浏览器通常会在段落前后自动添加外边距。

- **类型**: 块级元素（block-level）
- **分类**: 文本内容（text-content）
- **规范**: [HTML Living Standard - The p element](https://html.spec.whatwg.org/multipage/grouping-content.html#the-p-element)

## 属性

| 属性 | 源文件中的值 | 说明 |
|-----------|----------------|-------------|
| `class` | `"footer"` | 为元素指定一个或多个类名，用于 CSS 样式或 JavaScript 操作 |
| `id` | *(common)* | 为元素指定唯一标识符 |
| `style` | *(common)* | 行内 CSS 样式 |
| `title` | *(common)* | 元素的额外信息，通常以工具提示形式显示 |
| `lang` | *(common)* | 指定元素内容的语言 |
| `dir` | *(common)* | 指定元素文本的方向（从左到右或从右到左） |

## 子元素

该元素在源文件中仅包含纯文本内容 `© 2020 AGou . All Rights Reserved`，没有嵌套的子元素。

`<p>` 元素通常可以包含行内元素（phrasing content），例如：

| 子元素 | 在 `<p>` 中的用途 |
|---------------|------------------------|
| [`<a>`](a.md) | 在段落中嵌入超链接 |
| [`<span>`](span.md) | 对段落中的部分文本应用样式 |
| [`<strong>`](strong.md) | 标记重要文本 |
| [`<em>`](em.md) | 标记需要强调的文本 |
| [`<i>`](i.md) | 标记斜体文本 |

注意：`<p>` 元素**不能**包含其他块级元素（如 `<div>`、`<h1>`-`<h6>`、`<p>` 等），浏览器会自动关闭前一个 `<p>` 标签。

## 父元素

- [`<div>`](div.md)

在本源文件中，`<p>` 元素的父元素是 `<div class="main">`。

## 源文件示例

从源文件中提取的 `<p>` 元素实际使用代码：

```html
<!-- 来自 example/index.html -->
<p class="footer">© 2020 AGou . All Rights Reserved</p>
```

## 备注

- `<p>` 元素用于表示段落级别的文本，是语义化 HTML 的基础元素之一。
- 在本源文件中，`<p>` 被用作页脚版权信息容器，通过 `class="footer"` 应用页脚样式。
- 不推荐在 `<p>` 内部嵌套块级元素，浏览器解析时会自动提前关闭 `<p>` 标签，导致不可预期的布局结果。
- `<p>` 标签的结束标签 `</p>` 在某些情况下可以省略（当紧跟另一个块级元素时），但建议始终显式闭合以保持代码清晰。
