# <h1>

## 概述

`<h1>` 是 HTML 中最高级别的标题元素，用于表示页面或区块的主标题。在本源文件中，`<h1>` 包含文本"网址发布页"，作为整个页面的核心标题。

- **类型**：块级元素（block-level）
- **分类**：标题内容（heading content）
- **规范**：[HTML 规范 - Heading content](https://html.spec.whatwg.org/multipage/sections.html#the-h1,-h2,-h3,-h4,-h5,-and-h6-elements)

## 属性

本元素在源文件中没有使用任何属性。

| 属性 | 源文件中的值 | 说明 |
|-----------|----------------|-------------|
| `(common)` `class` | — | 用于为元素指定 CSS 类名（源文件中未使用） |
| `(common)` `id` | — | 用于为元素指定唯一标识符（源文件中未使用） |
| `(common)` `style` | — | 用于为元素指定内联样式（源文件中未使用） |

## 子元素

`<h1>` 元素在本源文件中仅包含纯文本内容"网址发布页"，不包含任何子元素。

这是文本内容元素，在当前源文件中未嵌套其他元素。

## 父元素

以下是本源文件中 `<h1>` 的父级元素：

- [`<div>`](div.md)

在源文件中，`<h1>` 位于 `<div class="main">` 内部。

## 源文件示例

从源文件中提取的 `<h1>` 元素使用示例：

```html
<!-- 来自 example/index.html -->
<h1>网址发布页</h1>
```

## 备注

- `<h1>` 是页面中最高级别的标题，每个页面通常只应使用一个 `<h1>` 元素以遵循语义化最佳实践。
- 标题元素（`<h1>` ~ `<h6>`）对于搜索引擎优化（SEO）和无障碍访问（accessibility）非常重要。
- 本源文件中 `<h1>` 与 [`<h2>`](h2.md) 配合使用，形成了合理的标题层级结构。
