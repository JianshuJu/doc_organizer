# `<h2>`

## 概述

`<h2>` 是 HTML 中的二级标题元素，用于表示文档结构中的第二层级标题。通常用于对页面内容进行分节，语义上从属于 `<h1>` 主标题。

- **类型**: 块级元素（block-level）
- **分类**: 标题内容（heading content）
- **规范**: [HTML 规范 - Heading Content](https://html.spec.whatwg.org/multipage/semantics.html#the-h1,-h2,-h3,-h4,-h5,-and-h6-elements)

## 属性

此元素在源文件中没有使用特定属性。

| 属性 | 源文件中的值 | 说明 |
|-----------|----------------|-------------|
| `class` (common) | — | 用于为元素指定 CSS 类名，便于样式控制 |
| `id` (common) | — | 为元素指定唯一标识符，用于锚点链接或 JavaScript 操作 |
| `style` (common) | — | 行内样式 |
| `lang` (common) | — | 指定元素内容的语言 |

## 子元素

此元素在源文件中仅包含纯文本内容，不包含子元素。

## 父元素

- [`<div>`](div.md)

## 源码示例

从源文件中提取的 `<h2>` 元素使用示例：

```html
<!-- 来自 example/index.html -->
<h2>请 Ctrl+D 收藏本页到浏览器收藏夹</h2>
```

## 备注

- `<h2>` 是语义化标签，有助于搜索引擎理解页面结构和内容层级。
- 每个页面应按逻辑顺序使用标题层级（`<h1>` → `<h2>` → `<h3>` ...），避免跳级使用。
- 在本源文件中，`<h2>` 位于 `<div class="content-top">` 容器内，用于提示用户收藏页面，是页面内容区的副标题。
